# Social features (share outfits, community feed)

**Labels:** `social`, `post-mvp`, `optional`

## Description

Phase 5.7: Share outfits with others; optional community inspiration feed. Requires profile visibility and careful privacy defaults.

## Tasks

- [ ] **Share**: "Share this outfit" → copy link, or share to social (Open Graph image + link). Outfit view may be public or unlisted.
- [ ] **Visibility**: User preference for "Show my outfit history" or "Public profile" (see profile visibility issue); default private.
- [ ] **Feed** (optional): Curated or user-submitted "outfit of the day" feed; like/save; link back to wardrobe-first flow.
- [ ] Moderation and safety: report, block; no PII in public views.
- [ ] Clearly optional; wardrobe-first flow remains primary.

## Acceptance criteria

- User can share an outfit via link or social; optional feed exists without compromising privacy defaults.
