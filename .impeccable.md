# OutfAI Design Context

## Design Context

### Users

Three equally-weighted personas, each arriving with a distinct headspace:

- **Student / young adult** — building a style identity on a budget, fashion-curious, time-pressed
- **Busy professional** — morning decision fatigue, wants fast reliable picks, efficiency matters
- **Fashion-forward creative** — uses OutfAI as a genuine creative tool, cares deeply about personal expression

The interface must respect all three: fast enough for the professional, expressive enough for the creative, accessible enough for the student. The unifying need is **reducing friction between "I have clothes" and "I feel good in what I'm wearing."**

### Brand Personality

Three words: **precise, edgy, considered**

The emotional target is _edgy and inspired_ — like stepping into a well-curated store where the edit itself signals taste. Not loud. Not aggressive. The sharpness is in the restraint.

The product should feel like it was built by people who care about craft — both in software and in clothing. Every choice is deliberate.

### Aesthetic Direction

**References:**

- **Arc'teryx / CELINE** — Premium restraint. High contrast. Typography-first. Luxury through ruthless editing. Nothing decorative that isn't structural.
- **Notion / Figma** — Productive and utility-forward. Non-corporate personality. The tool recedes so the work takes center stage.

The synthesis: a **high-end design tool with fashion editorial DNA**. Not a fashion brand's website (too art-directed, too slow). Not a generic SaaS dashboard (too corporate, no soul). The wardrobe is the content — the UI exists to surface it, not compete with it.

**Anti-references (what this should NOT look like):**

- AI-generated fashion apps with gradient text, glassmorphism cards, and pastel color palettes
- Generic productivity SaaS with rounded corners, blue primaries, and icon-above-heading card grids
- Streetwear brand sites with heavy texture, grunge type, and chaos-as-aesthetic

**Theme:** Support both light and dark. Light mode is the primary — fashion is consumed in daylight, on phones, during morning routines. Dark mode should feel equally deliberate, not just inverted.

**Current system note:** The existing "Cybersigilism" direction (sharp edges, signal orange `#ff4d00`, bone-white `#f4f3ef` / near-black `#0a0a0a`) has strong bones and is directionally aligned. The design is **open to rethinking** — but evolution should sharpen and refine the existing identity rather than abandon it. The zero-border-radius discipline and the restrained accent color strategy should be preserved.

**Typography:**

- **Body / UI / labels:** Hanken Grotesk (Google Fonts) — geometric grotesque, precise, contemporary. Loaded via `next/font/google` as `--font-hanken` → `--font-sans`.
- **Display / headings:** Bodoni Moda (Google Fonts) — classical high-contrast serif with strong italic cut, loaded via `next/font/google` as `--font-bodoni` → `--font-serif`. Always used with `italic` class for editorial character.
- **Note on italics:** Bodoni Moda has native italic variants — always apply `italic` to `font-serif` elements. The italic cut is the primary display voice.

### Design Principles

1. **The wardrobe takes center stage.** The UI is infrastructure, not decoration. Garment photography, outfit combinations, and user content are the visual heroes. Restrain the chrome.

2. **Luxury through editing.** Add nothing that doesn't earn its place. The signal orange accent works _because_ it's rare. Preserve that discipline across every element — every component, every animation, every piece of copy.

3. **Precision over personality.** When in doubt, tighter. More contrast. Sharper hierarchy. The edge should come from typographic precision and spatial confidence, not from visual tricks.

4. **Fast and honest.** Interactions should feel instant. No spinner theater, no loading decoration. The system respects the user's time. Motion is used for state changes, not atmosphere.

5. **Equally at home in a design tool and a boutique.** This product spans professional utility and personal expression. It should feel as disciplined as Figma and as considered as a well-curated store. Never sacrifice one for the other.
